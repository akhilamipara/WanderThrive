# WanderThrive - Travel Website

WanderThrive is a modern travel website that helps users explore destinations, plan trips, and book experiences. The website features a sleek UI, destination highlights, itinerary recommendations, and a client reviews slider powered by Swiper.js.

## 🚀 Features
- Stunning travel destinations showcase
- Easy-to-use booking system
- Detailed itineraries for various locations
- Responsive and mobile-friendly design
- Client reviews slider using Swiper.js

## 🛠️ Technologies Used
- **Frontend:** HTML, CSS, JavaScript
- **Styling:** Bootstrap, CSS animations

## 📌 Future Enhancements
- User authentication (login/signup)
- Personalized travel recommendations
- Payment gateway integration
- Interactive maps
